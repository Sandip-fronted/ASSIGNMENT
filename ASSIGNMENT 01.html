  • What are tags and attributes in HTML?

    • HTML tags are used to hold the HTML element. HTML element holds the content. HTML attributes are used to describe the characteristic of an HTML element in detail.


  • What are void elements in HTML? With Example.

    • All HTML elements are not required to contain a start tag and an end tag; these elements are referred to as void elements or empty elements since they lack both content and an end tag. These components are additionally known as unpaired tags.

  • What are HTML Entities? With Example

    • Invisible and reserved characters that would otherwise be read as HTML code are shown using an HTML entity. It is a line of text or a string that starts with a & and ends with a ;.


  • What are different types of lists in HTML? With Example.

  • unordered list — used to group a set of related items in no particular order.

  • ordered list — used to group a set of related items in a specific order.

  • description list — used to display name/value pairs such as terms and definitions.


  • What is the ‘class’ attribute in HTML? With Example.

    • One or more classnames are specified for an element by the class attribute. In a style sheet, the class attribute is typically used to identify a class.


  • What is the difference between the ‘id’ attribute and the ‘class’ attribute of HTML elements? With Example.

    • The basic difference between ID and Class is that the ID selector is applied only to one element in a page, whereas the class selector can be applied to several elements on a single page.
